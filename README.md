# Jurvis
[![996.icu](https://img.shields.io/badge/link-996.icu-red.svg)](https://996.icu)
#### What is Jurvis
Jurvis is a application for predict the football match result.

#### Evironment

```
Language:
-- python 3.6+
-- nodejs 8.0+
OS:
-- MacOS
```


#### Get Start

#### Install
```
//In home path yourpath:/Jurvis
sh install.sh

//In frontendpath yourpath:/Jurvis/fronted/
npm i
```

##### Run

```
//In home path yourpath:/Jurvis
sh run.sh
```
